# YouTube Spring Testing Module

This repository contains the complete source code for the **YouTube Spring Testing learning series**.

The goal of this module is to help developers:

- Understand JUnit fundamentals
- Master unit testing concepts
- Learn Spring Boot testing step by step
- Prepare for the **Spring Professional Certification**

## Structure

Each folder represents a single video in the series and contains:

- Source code examples
- Unit tests
- Notes related to the video topic

## Topics Covered

- JUnit Basics  
- JUnit Assertions  
- Test Lifecycle  
- Spring Boot Testing  
- Mocking and Integration Testing  
- Exam preparation tips  

More videos will be added continuously.
